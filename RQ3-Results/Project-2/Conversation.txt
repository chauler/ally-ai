WCAG Guideline violated: 1.4.5 Images of Text
Detection Algorithm: Checking for text over background images.
Explanation of Error: The header contains text over a background image. If the image fails to load or is not visible to a user with visual impairments, the text may not be perceivable. Consider adding a solid background color as a fallback.

</div>

Detection Algorithm: Checking for img tags without alt attributes.
Explanation of Error: The img tag is missing an alt attribute. This attribute is important for users who can't see the image. It provides a textual description of the image.
